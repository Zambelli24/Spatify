Spatify API

This is Spatify, the Spotify API companion that allows you to make more direct and specific queries about your favorite artists.

    http://localhost:5000/search_artist?artist=(your query)

  • If you wish to search for an artist, you must first find the artist id that Spotify assigns to them
    • In order to do this, pass in the name of the artist you wish to find and one of two things will happen:
      • you will receive a list of artists that match your search
      • you will find your desired artist

    http://localhost:5000/related_artists?artist=(your query)

  • If you have an artist in mind and would like to find related artists,
    • simply pass the name of the artist as the search parameter and receive a list of related artists

    http://localhost:5000/track_search?artist=(your query)

  • If you wish to search for all of the tracks by a particular artist,
    • pass the name of the artist as the search parameter and receive a redirection url to paste into the search bar
    • This url will give you all of the songs by that artist in pages consisting of ten items
